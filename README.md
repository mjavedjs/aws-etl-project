# AWS ETL Project

This project demonstrates an ETL pipeline built using **AWS Glue (Visual ETL)**, **Lambda**, **S3**, and **Athena**.

## Workflow
1. Extract data from S3
2. Transform using AWS Glue Visual ETL
3. Load transformed data back to S3
4. Trigger ETL automatically using AWS Lambda
5. Query processed data using AWS Athena
